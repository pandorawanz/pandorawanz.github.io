html运行方法

HTML运行方法

1.浏览器打开
2.http-server

http-server

打开终端或者 gitbash，输入
npm install -g http-server
即安装成功（如果提示命令不存在，需要先下载安装 nodejs），安装后切换到对应目录，启动静态服务器，在html目录下输入http-server

命令行翻译工具

打开终端或者 gitbash，输入
npm install -g fanyi
然后在终端下输入fanyi 你好
即可进行及时翻译
